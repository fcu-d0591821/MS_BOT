# MS_BOT

## 簡介
- 一隻只會在新手村打小石球的 BOT
- 然後還打不太到

## 使用
```
python -m venv venv
\script\active
pip install -r requirement.txt
python main.py
```
- 按 F2 開始或暫停

## 可能需要改的東西
- mob.png: 小石球的圖片
- player.png: 角色的圖片
